Steps to install jenkins in jenkins server
1>"wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -"
2>"sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'"
3>"sudo apt update"
4>"sudo apt install jenkins"
5>"sudo systemctl start jenkins"
6>"sudo systemctl status jenkins"
Login to jenkins
1>Managejenkins=>ManagePlugins=>Available=>Search"SSH agent"=>install without restart=>after installiation=>"click the checkbox for Restart jenkins" 
2>Login to jenkins with your credentials
